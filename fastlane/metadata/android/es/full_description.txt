Esta es una aplicación no oficial de Android para el traductor DeepL. Utiliza una tecnología llamada WebView, un componente GUI de Android, para que parezca una aplicación nativa. Se utiliza JavaScript para ocultar todos los elementos excepto los más necesarios.

<b>Características:</b>

* Muestra sólo la pantalla de traducción mediante WebView
* Cambiar al modo oscuro
* Lanza DeepL desde la acción de selección de texto flotante
* Aceda ao ecrã de definições premindo e mantendo premido o ícone da aplicação
